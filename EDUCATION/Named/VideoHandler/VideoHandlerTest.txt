СДЕЛАТЬ ТЕКСТОВОЕ ПОЛЕ (МНОГОСТРОЧНОЕ) НЕ ВИДИМЫМ - ПРЕДЛОЖЕНИЕ ПО ЗАПРОСУ В НЕТ:
Как сделать так, чтобы моя текстовая область была скрыта?
Все дают вам ответы, но не объясняют причины. Вот вам ответ: если вы используете правило CSS visibility:hidden; область текста будет невидимой, но всё равно будет занимать место. Если вы используете правило CSS display:none;Текстовое поле будет скрыто,ионо не будет занимать место на экране — другими словами, не будет пробелов там, где оно должно было быть. Ниже приведен наглядный пример.
Итак, вы хотите, чтобы что-то подобное былополностьюскрыто:
<textarea cols="20" rows="20" style="display:none;">
Пример
/* no styling should show up for either method */
textarea {
  background: lightblue;
  border: 1px solid blue;
  font-weight: bold;
}
<p><strong>Textarea (not hidden)</strong></p>
<textarea>Text within.</textarea>
<p>Some text after.</p>
<hr />
<p><strong>Textarea with <code>display:none;</code></strong></p>
<textarea style="display:none;">Text within.</textarea>
<p>Some text after. Neither height nor margin/padding/layout kept. No other styles visible.</p>
<hr />
<p><strong>Textarea with <code>visibility:hidden;</code></strong></p>
<textarea style="visibility:hidden;">Text within.</textarea>
<p>Some text after. Height and margin/padding/layout kept. No other styles visible.</p>

Еще по теме скрыть элемент (в данном случае по нажатии кнопки):

JavaScript изменяет свойство display элемента на none, тем самым скрывая его. 
HTML:
Код

<button id="hideButton">Спрятать</button>
<div id="myElement">Этот элемент будет скрыт</div>
JavaScript:
JavaScript

const hideButton = document.getElementById('hideButton');
const myElement = document.getElementById('myElement');

hideButton.addEventListener('click', () => {
  myElement.style.display = 'none';
});
CSS (необязательно, но рекомендуется для управления видимостью):
Код

#myElement {
  display: block; /* Или любое другое значение по умолчанию */
}
Пошаговое объяснение:
1. HTML:
Создаем кнопку с идентификатором hideButton и элемент (например, div), который будет скрываться, с идентификатором myElement.
2. JavaScript:
Получаем ссылки на кнопку и элемент.
Добавляем обработчик события click к кнопке.
В обработчике события меняем свойство display элемента на none.
3. CSS (дополнительно):
Устанавливаем начальное значение display элемента (например, block), чтобы он был виден по умолчанию.
Когда пользователь нажимает кнопку, JavaScript изменит свойство display элемента на none, и элемент будет скрыт. Он больше не будет занимать место на странице, по словам Habr. Если нужно показать элемент обратно, то нужно изменить свойство display на block или на любое другое значение, которое было установлено изначально.
